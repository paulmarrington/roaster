General

uSDLC2 requires a lightweight modular server that can provide streaming as well as HTTP support. Node-JS fits the bill with a little tweaking. The server must support coffee-script and LESS out-of-the box and be installed ready to run with a single command. It must be extensible with serving extensions based on file type (MIME). Finally it must be able to transparently integrate into a supporting project that requires a server.

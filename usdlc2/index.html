Roaster

Yet another server for a browser based client. More a collections of conveniences than a framework - can be used with any of the node based frameworks.

Preface

Installation

Roaster and roaster based servers have a single-script install - well one for Windows and one for Unix-like systems.

Communications

Roaster provides different communications mechanisms for different needs and targets - including http and pub-sub (faye), web-sockets and basic sockets.
